# zenn-content

[Zenn](https://zenn.dev/yukamiya) 公開用のコンテンツを管理するリポジトリです。Zenn CLI で Markdown を管理・プレビューします。

## Setup

```bash
npm install
```

## 使い方

```bash
# 記事のプレビュー
npx zenn preview

# 新しい記事を作成
npx zenn new:article

# 新しい本を作成
npx zenn new:book
```

詳細は [Zenn CLI を使ってみる](https://zenn.dev/zenn/articles/install-zenn-cli) を参照してください。
